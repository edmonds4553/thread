### 并发

&emsp;同一个对象被多个线程同时操作，一个处理器同时处理多个任务

### 死锁

&emsp;多个线程同时被阻塞，它们中的一个或者全部都在等待某个资源被释放。由于线程被无限期地阻塞，因此程序不可能正常终止

### 实现多线程几种方式

* 继承Thread类
* 实现Runnable接口
* 实现Callable和FutureTask接口
* 通过线程池创建
